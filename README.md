# 💫 About Me

👋 Hi there! I'm **Miguel Álvarez**, a self-taught Frontend Developer passionate about crafting scalable, performant, and visually stunning web experiences.

🚀 I specialize in:
- **React** & **Next.js** – Building modern, high-performance web apps.
- **Tailwind CSS** – Creating responsive, clean, and flexible layouts.
- **GSAP** & **Framer Motion** – Designing smooth animations and interactive UIs.
- **Three.js** – Exploring 3D on the web to build immersive interfaces.
- **Prismic** – Implementing scalable CMS systems for dynamic content management.

🧠 **Currently Learning & Expanding Into**:
- **TypeScript** – Enhancing code safety and maintainability.
- **Vitest & Jest** – Strengthening testing practices.
- **React 19** – Exploring new experimental features and hooks.
- **Zustand** – Simplifying state management in modern React apps.
- **AI Tools** – I'm actively taking courses and applying AI to boost productivity, automate workflows, and support project ideation & execution.

📌 I'm also focused on applying clean code practices, scalable component design, and performance-first development to every project I touch.
